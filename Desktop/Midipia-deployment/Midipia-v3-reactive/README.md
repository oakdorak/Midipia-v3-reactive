# Midipia v3: Reactive 🎹🌬️

The most visceral version of Midipia, featuring "breathing" reactivity driven by MIDI Aftertouch.

## 🚀 How to Run
1. Clone this repository.
2. Open `index.html` in a MIDI-supported browser (Google Chrome or Microsoft Edge are recommended).
3. Connect your MIDI controller.
4. Play!

## ⚙️ Requirements
- **Browser:** Chrome, Edge, or any browser supporting the **Web MIDI API**.
- **Hardware:** A MIDI controller that supports **Aftertouch** (pressure sensitivity).
- **Connection:** USB or MIDI-to-USB interface.

## 🎨 Visual Logic
This version implements **Reactive Modulation**. While notes still trigger colors, the *intensity* of the visual experience is controlled by your fingertips. By applying pressure (aftertouch) to the keys, you modulate the brightness and saturation of the canvas, making the visuals "breathe" in sync with your performance.
