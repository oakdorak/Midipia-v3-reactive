# Midipia v2: Polychrome 🎹🎨

A dynamic MIDI visualizer that layers colors in real-time, creating a shifting, polychromatic canvas.

## 🚀 How to Run
1. Clone this repository.
2. Open `index.html` in a MIDI-supported browser (Google Chrome or Microsoft Edge are recommended).
3. Connect your MIDI controller.
4. Play!

## ⚙️ Requirements
- **Browser:** Chrome, Edge, or any browser supporting the **Web MIDI API**.
- **Hardware:** Any MIDI-compatible keyboard or controller.
- **Connection:** USB or MIDI-to-USB interface.

## 🎨 Visual Logic
Unlike the minimalist version, Polychrome uses **Dynamic Mixing**. As you play notes, colors are added to a queue and blended into a fluid linear gradient. The result is a living, breathing tapestry of color that evolves based on your chord progressions.
